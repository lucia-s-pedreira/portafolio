SQL Data Analysis

Proyecto de análisis de usuarios activos y resumen de pedidos con SQL.


-- Obtener usuarios activos
SELECT id, nombre, email
FROM usuarios
WHERE activo = 1
ORDER BY fecha_registro DESC;

-- Resumen de pedidos
SELECT MONTH(fecha_pedido) AS mes, COUNT(*) AS total_pedidos
FROM pedidos
GROUP BY MONTH(fecha_pedido);

import csv, requests

response = requests.get('https://api.ejemplo.com/datos')
data = response.json()

with open('salida.csv', 'w', newline='') as f:
    writer = csv.DictWriter(f, fieldnames=data[0].keys())
    writer.writeheader()
    writer.writerows(data)

Machine Learning

Modelo de clasificación con Random Forest para datos de Iris.


from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score

X, y = load_iris(return_X_y=True)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3)
modelo = RandomForestClassifier(n_estimators=100).fit(X_train, y_train)
print("Precisión:", accuracy_score(y_test, modelo.predict(X_test)))
